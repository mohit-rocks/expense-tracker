# Expense Tracker Application
This is a simple application built for learning and exploring symfony.

Features:
- Import the expenses from the CSV.
- Allows to view all the expense and filter based on types.
- Analyze the expense.
- Predict the pattern based on your current expenses. AI is used.
- Various reports available for the expenses.

## Tech Stack:
- Symfony
- Tailwind
- PostgreSQL database
